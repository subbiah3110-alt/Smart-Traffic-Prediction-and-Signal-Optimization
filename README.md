# Smart Traffic Prediction and Signal Optimization

## Project Overview
This project predicts traffic density (Low, Medium, Heavy) from traffic images using Deep Learning and dynamically adjusts traffic signal timings.

## Features
- Traffic density classification
- Deep learning-based prediction
- Automatic traffic signal timing optimization
- Real-time decision support

## Technologies Used
- Python
- TensorFlow / Keras
- MobileNet
- NumPy

## Traffic Signal Timing
| Traffic Density | Green Signal | Red Signal |
|----------------|-------------|-----------|
| Low            | 20 sec      | 40 sec    |
| Medium         | 30 sec      | 30 sec    |
| Heavy          | 40 sec      | 20 sec    |

## How to Run
1. Install dependencies
2. Run the training script
3. Load the trained model
4. Predict traffic density from input images
